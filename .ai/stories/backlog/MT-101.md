# MT-101: Project Scaffolding

## Description

Create base project structure with TypeScript, Vite, React, Express, and Tailwind CSS configured.

## Acceptance Criteria

- [ ] npm project initialized with all dependencies
- [ ] TypeScript configured for client and server
- [ ] Vite configured for React + Tailwind
- [ ] Folder structure created: `src/{client,server,shared}/`
- [ ] Dev scripts added to package.json
- [ ] `npm run dev` starts both Vite dev server and Express server
- [ ] React app shows "Hello World" at `http://localhost:5173`
- [ ] Express responds to `GET /api/status` at `http://localhost:3001`

## Technical Notes

### Dependencies to Install

**Production:**
- express
- socket.io
- multer
- uuid

**Development:**
- typescript
- vite
- react, react-dom
- @vitejs/plugin-react
- tailwindcss, postcss, autoprefixer
- @types/node, @types/express, @types/uuid
- concurrently (run client + server together)
- nodemon (auto-restart server)
- tsx (run TypeScript directly)

### Scripts

```json
{
  "dev": "concurrently \"npm run dev:client\" \"npm run dev:server\"",
  "dev:client": "vite",
  "dev:server": "nodemon --exec tsx src/server/index.ts",
  "build": "vite build && tsc -p tsconfig.server.json",
  "start": "node dist/server/index.js"
}
```

### Folder Structure

```
src/
├── client/
│   ├── App.tsx
│   ├── main.tsx
│   └── index.css
├── server/
│   └── index.ts
└── shared/
    └── types.ts
```

## Files to Create

- `package.json` - Project manifest with scripts
- `tsconfig.json` - Client TypeScript config
- `tsconfig.server.json` - Server TypeScript config
- `vite.config.ts` - Vite configuration
- `tailwind.config.js` - Tailwind configuration
- `postcss.config.js` - PostCSS configuration
- `src/client/App.tsx` - React app entry
- `src/client/main.tsx` - React DOM render
- `src/client/index.css` - Tailwind imports
- `src/server/index.ts` - Express server entry
- `src/shared/types.ts` - Shared TypeScript types
- `index.html` - Vite HTML entry
- `.gitignore` - Ignore node_modules, uploads, dist
