# MT-109: Main App Layout

## Description

Compose FileUpload and FileList components into main app UI with header and status indicators.

## Acceptance Criteria

- [ ] `App.tsx` layout with header, upload section, file list
- [ ] App header with "Mobile Transporter" title
- [ ] Connection status indicator (green/red dot)
- [ ] Footer showing connected device count
- [ ] Clean, intuitive single-page layout
- [ ] Upload zone at top, file list below
- [ ] Error boundary for unhandled errors
- [ ] Responsive design (mobile-first)

## Technical Notes

### Layout Structure

```tsx
function App() {
  const { connected, deviceCount } = useSocket();
  const { files, deleteFile } = useFileSync();

  return (
    <div className="min-h-screen bg-gray-50">
      <Header connected={connected} />
      <main className="container mx-auto px-4 py-8 max-w-2xl">
        <FileUpload />
        <div className="mt-8">
          <h2 className="text-lg font-semibold mb-4">Shared Files</h2>
          <FileList files={files} onDelete={deleteFile} />
        </div>
      </main>
      <Footer deviceCount={deviceCount} />
    </div>
  );
}
```

### Header Component

```tsx
function Header({ connected }: { connected: boolean }) {
  return (
    <header className="bg-white shadow-sm px-4 py-3">
      <div className="container mx-auto max-w-2xl flex items-center justify-between">
        <h1 className="text-xl font-bold">Mobile Transporter</h1>
        <div className="flex items-center gap-2">
          <span className={`w-2 h-2 rounded-full ${connected ? 'bg-green-500' : 'bg-red-500'}`} />
          <span className="text-sm text-gray-600">
            {connected ? 'Connected' : 'Disconnected'}
          </span>
        </div>
      </div>
    </header>
  );
}
```

### Footer Component

```tsx
function Footer({ deviceCount }: { deviceCount: number }) {
  return (
    <footer className="fixed bottom-0 left-0 right-0 bg-white border-t px-4 py-2">
      <div className="container mx-auto max-w-2xl text-center text-sm text-gray-500">
        {deviceCount} device{deviceCount !== 1 ? 's' : ''} connected
      </div>
    </footer>
  );
}
```

### Device Count Tracking

Add to useSocket hook:
```typescript
const [deviceCount, setDeviceCount] = useState(0);

socket.on('device:joined', () => setDeviceCount(c => c + 1));
socket.on('device:left', () => setDeviceCount(c => Math.max(0, c - 1)));
```
