# MT-103: File Upload REST API

## Description

Expose HTTP endpoint for file uploads using Multer middleware.

## Acceptance Criteria

- [ ] Multer middleware configured for multipart uploads
- [ ] `POST /api/files` endpoint accepts file uploads
- [ ] Request validation (returns 400 if no file)
- [ ] Response returns file metadata as JSON with 201 status
- [ ] Error handling for file too large (100MB limit)
- [ ] Can upload file via curl/Postman

## Technical Notes

### Multer Configuration

```typescript
const upload = multer({
  dest: 'uploads/',
  limits: { fileSize: 100 * 1024 * 1024 }, // 100MB
});
```

### Endpoint

```
POST /api/files
Content-Type: multipart/form-data
Body: file (file field)

Response 201:
{
  "id": "uuid",
  "name": "original-filename.pdf",
  "size": 12345,
  "uploadedAt": "2026-01-11T12:00:00Z"
}

Response 400:
{ "error": "No file provided" }

Response 413:
{ "error": "File too large" }
```
